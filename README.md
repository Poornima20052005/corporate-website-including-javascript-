Corporate Company Website with JavaScript
This project is a responsive corporate business website built using HTML, CSS, and JavaScript.
It represents a professional company profile and includes interactive UI features to improve user experience.

Features: Responsive layout for mobile and desktop
          Modern corporate design with images
          Services and projects showcase
          Animated company statistics counter
          Mobile navigation toggle menu
          Smooth scrolling navigation button
          Careers section with Apply interaction
          Contact form with success feedback
          Hover effects, shadows, and clean layout

Technologies Used: HTML5,CSS3,JavaScript (DOM manipulation, events, animations)
Responsive Design (Flexbox, Media Queries)

Purpose of the Project
This project was created to practice front-end web development and understand how real corporate websites include interactive features using JavaScript. It serves as a portfolio project for placements, internships, and interviews.# corporate-website-including-javascript-
